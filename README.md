# Pokedex

Aplicação web interativa que consome a PokeAPI para exibir informações de Pokémons de forma dinâmica, com interface moderna e cards expansíveis.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Consumo de API REST (PokeAPI)

---

## Funcionalidades

* Listagem de Pokémons com paginação
* Consumo de dados em tempo real via API
* Cards interativos com expansão ao clique
* Exibição de detalhes:

  * Número
  * Nome
  * Tipo(s)
  * Altura (Height)
  * Peso (Weight)
  * Habilidades (Abilities)
* Interface responsiva e estilizada por tipo de Pokémon

---

## Aprendizados

Este projeto permitiu praticar:
